DESARROLLO Y ARQUITECTURAS WEB
Virginia Martinez
4-A. Turno Noche
Este repositorio corresponde a una actividad inicial orientada a la introducción en el uso de Git y GitHub como herramientas de control de versiones. A lo largo del trabajo se desarrollan los conceptos fundamentales necesarios para la gestión de proyectos, incluyendo la instalación y configuración de Git, la creación y vinculación de repositorios locales y remotos, y el versionado de archivos mediante commits.
