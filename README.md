🧾 Descripción del proyecto

Este repositorio corresponde a una actividad inicial orientada a la introducción en el uso de Git y GitHub como herramientas de control de versiones.

A lo largo del trabajo se desarrollan los conceptos fundamentales necesarios para la gestión de proyectos, incluyendo:

Instalación y configuración de Git
Creación de repositorios locales
Vinculación con repositorios remotos en GitHub
Uso del versionado de archivos mediante commits

🎯 Objetivo

El objetivo principal de esta actividad es familiarizarse con el flujo de trabajo básico de control de versiones, comprendiendo la importancia de Git en el desarrollo de software colaborativo.
